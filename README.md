#OG Images for Shopware

- Shopping Worlds
- Categories
- Articles


To install, just download and unzip Archive into custom/plugins. After that use Shopware's Pluginmanager 
to install properly.


CLI command to generate Article's Images: "mc:ogimages:article:full"
